📘 Class 10 Maths Formula Sheet

All important formulas of Class 10 Maths in one place.

📐 Chapter 1: Real Numbers

  • Euclid’s Division Lemma: a = bq + r
  • HCF × LCM = Product of two numbers
  • For rational number p/q, decimal terminates if denominator has form 2ⁿ × 5ᵐ

📊 Chapter 2: Polynomials

  • Quadratic polynomial: ax² + bx + c
  • Zeros of polynomial = values of x where p(x)=0
  • Sum of zeros = -b/a
  • Product of zeros = c/a

📈 Chapter 3: Pair of Linear Equations

  • General form: a₁x + b₁y + c₁ = 0
  • Solution methods: substitution, elimination, cross-multiplication

📐 Chapter 4: Quadratic Equations

  • Standard form: ax² + bx + c = 0
  • Quadratic formula: x = (-b ± √(b² - 4ac)) / 2a
  • Discriminant: D = b² - 4ac
  • If D > 0 → 2 real roots
  • If D = 0 → equal roots
  • If D < 0 → no real roots

📐 Chapter 5: Arithmetic Progression (AP)

  • nth term: aₙ = a + (n-1)d
  • Sum of n terms: Sₙ = n/2 [2a + (n-1)d]

📊 Chapter 6: Triangles

  • Area = 1/2 × base × height
  • Similarity ratio rules apply
  • Basic Proportionality Theorem (BPT)

📐 Chapter 7: Coordinate Geometry

  • Distance formula: √[(x₂-x₁)² + (y₂-y₁)²]
  • Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
  • Area of triangle using coordinates

📊 Chapter 8: Trigonometry

  • sin θ = opposite / hypotenuse
  • cos θ = adjacent / hypotenuse
  • tan θ = opposite / adjacent
  • sin²θ + cos²θ = 1
  • 1 + tan²θ = sec²θ
  • 1 + cot²θ = cosec²θ

📐 Chapter 9: Circles

  • Area = πr²
  • Circumference = 2πr
  • Tangent is perpendicular to radius

📊 Chapter 10: Mensuration

  • Cube volume = a³
  • Cuboid volume = l × b × h
  • Sphere volume = 4/3 πr³
  • Cylinder volume = πr²h

📐 Chapter 11: Statistics

  • Mean = sum of observations / number of observations
  • Median = middle value
  • Mode = most frequent value

📊 Chapter 12: Probability

  • Probability = Favorable outcomes / Total outcomes
  • Range: 0 to 1
